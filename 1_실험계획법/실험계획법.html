Hypothesis Testing Framework

1) If the variances are know

\[ If \; \sigma=\sigma_{1}=\sigma_{2}=0.3 \qquad H_{0}:\mu_{1}=\mu_{2}, \ H_{1}:\mu_{1}\neq\mu_{2} \qquad Z_{0} = \frac{\overline{y_{1}}-\overline{y_{2}}}{\sqrt{\frac{\sigma_2^1}{n_{1}}+{\frac{\sigma_2^2}{n_{2}}}}} \]

Test statistics: Z0 (two-side test)

If sigma = sigma1 = sigma2 = 0.3

  1. 기각역을 통한 검정 |Z(alpha/2)| < |Z0| 이면 기각 (결과: H0 기각)

Case1: Large Sample Size

\[ H_{0}:\mu_{1}=\mu_{2}, \ H_{1}:\mu_{1}\neq\mu_{2} \qquad Z_{0} = \frac{\overline{y_{1}}-\overline{y_{2}}}{\sqrt{\frac{S_2^1}{n_{1}}+{\frac{S_2^2}{n_{2}}}}} \]

n > 30 => Sample sizes were large enough => Z-test

Case2 1: Small Sample Size, \(\sigma_1^2 = \sigma_2^2\) (크기가 작고 모분산이 같은 경우)

\[ t_{0} = \frac{\overline{y_{1}}-\overline{y_{2}}}{S_{p}\sqrt{\frac{1}{n_{1}}+{\frac{1}{n_{2}}}}} \qquad S_p^2=\frac{{(n_1-1)}S_1^2-(n_2-1)S_2^2}{n_1+n_2-2} \qquad v=n_1+n_2-2 \]

  1. 기각역을 통한 검정 |t(alpha/2)| < |t0| 이면 기각 (결과: H0 기각)

Completely Randomized Design (CRD)

  • a levels or treatments

  • n replicates

  • Run in random order

  • There are fixed effects case and the random effects case.

  • The goal is to test hypotheses about the equality of the 𝑎treatment means

One Way ANOVA model

  • effects model: \(\mu_{ij} = \mu + \tau_i + \epsilon_{ij}\)

  • Formal Statistical Hypothesis: \(H_0:\tau_1=…=\tau_a=0, \ H1:At \ least \ one \ is \ different\)

  • Decomposition of the Total Sum of Squares (SS):

    \(\sum_{i=1}^a\sum_{j=1}^n(y_{ij}-\overline{y}_{..})^2 = n\sum_{i=1}^a(y_{i.}-\overline{y}_{..})^2 + \sum_{i=1}^a\sum_{j=1}^n(y_{ij}-\overline{y}_{i.})^2 \\SS_T = SS_{Treatments} + SS_E\)

  1. aov를 이용해서 CRD 분석(권장)
